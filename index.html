Backup & Restaurar Dados

Exporte seus candidatos para um arquivo JSON e importe quando precisar — mesmo em outro dispositivo ou versão do sistema.

Exportar (salvar backup)

Baixa um arquivo .json com todos os candidatos. Guarde-o em local seguro.

Importar (restaurar backup)

Selecione um arquivo .json exportado anteriormente. Os dados serão mesclados com os atuais (não apaga nenhum).

Recuperar dados do arquivo anterior

Se você ainda tem o arquivo HTML antigo, abra-o no navegador, vá ao ADM → Backup → Exportar. Depois importe aqui o arquivo gerado.

Passo a passo:
1⃣ Abra o arquivo HTML antigo no navegador
2⃣ Entre na área ADM → clique em 💾 Backup
3⃣ Clique em Baixar backup
4⃣ Volte neste arquivo e clique em Importar

Configuração (só 1 vez — 5 minutos)
1
Acesse emailjs.com → Sign Up Free → crie conta com seu Gmail ([email protected])
2
No painel: Email Services → Add New Service → escolha Gmail → conecte sua conta → copie o Service ID
3

Subject: {{subject}}
To Email: {{to_email}}
To Name: {{to_name}}
Body (HTML): {{{message}}}
Use {{{message}}} com 3 chaves para renderizar HTML. Salve e copie o Template ID
